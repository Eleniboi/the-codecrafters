### Go maps

a map is use to store data value in key

the length of a map is equal the number of element in the map 
And the default value(zero value) of a map is nil

### declaring a map

1. using the var keyword e.g
```go
var sam = map[string]string{"hobbie": "coding", "obsession": "God"}
```
2. using the := sign
```go
sam := map[string]string{"hobbie": "coding", "obsession": "God"}
```
