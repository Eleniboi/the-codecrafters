
## Go Variables

+ go variable are container which store or hold data value

To use a variable in go you have to declare it using one : 
1. The "var" keyword 

example
```go
var samuel string
```
2.The short variable declaration(walrus operator)

example 
```go
samuel := "value"
```
NOTE: here in the walrus operator you have to assign a value on that same line the variable was declared.
